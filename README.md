# Practica 1

## Descripción

Esta es la **Práctica 1**. Consiste en un **proyecto vacío** creado en Android Studio cuyo objetivo principal es familiarizarse con:

- El entorno de desarrollo integrado (IDE) **Android Studio**
- La **estructura básica** de un proyecto Android
- Los archivos y carpetas fundamentales que componen una aplicación

> **Nota:** Esta práctica no contiene funcionalidades específicas ni lógica de negocio. Es un proyecto base para comprender la organización de los recursos, el manifiesto, los gradle files y la jerarquía de carpetas.

## Cómo ejecutar el proyecto
1. Clona el repositorio:
   
   ```bash
   git clone https://github.com/valmejia/TDSMH_P1P6_VMG.git

2. Abre Android Studio.
3. Espera a que se sincronicen las dependencias (Gradle).
4. Conecta un dispositivo físico o inicia un emulador.
5. Haz clic en Run.
